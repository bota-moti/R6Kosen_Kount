## プロジェクトについて

令和6年度高専祭においてクラス杯投票を一般の人に行ってもらうために開発したシステム
当日はタッチモニターを使用することでＰＣに疎い人でも投票することができるようにした
なぜFormsではなく、システム開発することになったのか
今年の高専祭では色々な新しいことに挑戦するというテーマでした。
